# Python & MySQL Database Connection Tutorial

This repository contains a simple, beginner-friendly Python script designed to establish a connection with a local MySQL server and list all available databases. 

## 🛠️ Prerequisites

Before running the script, make sure you have the following installed on your local machine:

1. **Python 3.x**
2. **MySQL Server** (via XAMPP, WampServer, or MySQL Community Server)
3. **MySQL Connector for Python**: You can install it using pip:
   ```bash
   pip install mysql-connector-python
'''
