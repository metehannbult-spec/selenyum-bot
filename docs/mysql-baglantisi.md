# 🤖 Selenium Bot

Bu proje, Selenium kütüphanesini öğrenme sürecimde geliştirdiğim otomasyon çalışmalarını ve notlarımı içermektedir.

## 🎯 Projenin Amacı

Bu depo, Selenium ile web otomasyonu geliştirmeyi öğrenirken yaptığım çalışmaları düzenli bir şekilde belgelemek amacıyla oluşturulmuştur.

Bu proje içerisinde;

- Selenium kurulumu
- Chrome WebDriver kullanımı
- Chrome ayarlarının yapılandırılması
- MySQL veritabanı bağlantısı
- Okul otomasyonu örneği
- Selenium ile web otomasyonu

konularını uygulamalı olarak öğreniyorum.

# 🚀 Nasıl Çalıştırılır?

1. Depoyu klonlayın.

```bash
git clone https://github.com/metehannbulut-spec/selenyum-bot.git
```

2. Gerekli kütüphaneleri yükleyin.

```bash
pip install selenium
```

3. ChromeDriver'ın bilgisayarınızda kurulu olduğundan emin olun.

4. Python dosyalarını çalıştırın.
